A node can have three different types of attributes. An attribute containing another node, an attribute containing an array of other nodes, or an attribute containing a primitive value such as a boolean, string, or number.

Use grasp --help syntax for an overview of the syntax, and grasp --help node-type (eg. grasp --help if) for more about information about a node type.

Skip to statements, expressions, declarations, clauses, or categories of node types.

Based on the Mozilla SpiderMonkey AST format.

Misc

program (Program)

node array attributes:
body

note:

The root node of a JavaScript program's AST.

ident (Identifier)

primitive attributes:
name

example:

{% highlight javascript %} x {% endhighlight %}

literal (Literal)

primitive attributes:
value (val)

examples:

{% highlight javascript %} true 1 "string" {% endhighlight %}

regex (RegExpLiteral)

primitive attributes:
regex

example:

{% highlight javascript %} /^sh+/gi {% endhighlight %}

prop (Property)

node attributes:
key,  value (val)
primitive attributes:
kind

syntax:

key: value

example:

{% highlight javascript %} a: 1 {% endhighlight %}

note:

An object expression (obj) has a list of properties, each being a property.

spread (SpreadElement)

node attributes:
argument (arg)

template-element (TemplateElement)

primitive attributes:
tail,  value (val)

Statements

empty (EmptyStatement)

example:

{% highlight javascript %} ; {% endhighlight %}

block (BlockStatement)

node array attributes:
body

syntax:

{
  statement_1
  statement_2
  ...
  statement_n
}

example:

{% highlight javascript %} { x = 1; f(); x++; } {% endhighlight %}

exp-statement (ExpressionStatement)

node attributes:
expression (exp)

syntax:

expression;

example:

{% highlight javascript %} 2; {% endhighlight %}

note:

When an expression is used where a statement should be, it is wrapped in an expression statement.

if (IfStatement)

node attributes:
test,  consequent (then),  alternate (alt, else)

syntax:

if (test)
  consequent
[else
  alternate]

examples:

{% highlight javascript %} if (even(x)) { f(x); } if (x === 2) { x++; } else { f(x); } {% endhighlight %}

label (LabeledStatement)

node attributes:
label,  body

syntax:

label: body;

example:

{% highlight javascript %} outer: for (i = 0; i < xs.length; i++) { for (j = 0; j < ys.length; j++) { if (xs[i] === ys[j]) { break outer; } } } {% endhighlight %}

break (BreakStatement)

node attributes:
label

syntax:

break [label];

examples:

{% highlight javascript %} break; break outer; {% endhighlight %}

continue (ContinueStatement)

node attributes:
label

syntax:

continue [label];

examples:

{% highlight javascript %} continue; continue outerLoop; {% endhighlight %}

with (WithStatement)

node attributes:
object (obj),  body

syntax:

with (object)
  body

example:

{% highlight javascript %} with ({x: 42}) { f(x); } {% endhighlight %}

switch (SwitchStatement)

node attributes:
discriminant
node array attributes:
cases

syntax:

switch (discriminant) {
  case_1
  case_2
  ...
  case_n
}

example:

{% highlight javascript %} switch (num) { case 1: f('one'); break; case 2: f('two'); break; default: f('too many'); } {% endhighlight %}

return (ReturnStatement)

node attributes:
argument (arg)

syntax:

return argument;

example:

{% highlight javascript %} return f(2); {% endhighlight %}

throw (ThrowStatement)

node attributes:
argument (arg)

syntax:

throw argument;

example:

{% highlight javascript %} throw new Error("oops"); {% endhighlight %}

try (TryStatement)

node attributes:
block,  handler,  finalizer

syntax:

try
  block
[handler]
[finally
   finalizer]

example:

{% highlight javascript %} try { result = parse(input); } catch (error) { console.error(error.message); result = ''; } finally { g(result); } {% endhighlight %}

while (WhileStatement)

node attributes:
test,  body

syntax:

while (test)
  body

example:

{% highlight javascript %} while (x < 2) { f(x); x++; } {% endhighlight %}

do-while (DoWhileStatement)

node attributes:
test,  body

syntax:

do
  body
while (test);

example:

{% highlight javascript %} do { f(x); x++; } while (x < 2); {% endhighlight %}

for (ForStatement)

node attributes:
init,  test,  update,  body

syntax:

for ([init]; [test]; [update])
  body

example:

{% highlight javascript %} for (let x = 0; x < 2; x++) { f(x); } {% endhighlight %}

for-in (ForInStatement)

node attributes:
left (l),  right (r),  body

syntax:

for (left in right)
  body

example:

{% highlight javascript %} for (let prop in object) { f(object[prop]); } {% endhighlight %}

for-of (ForOfStatement)

node attributes:
left (l),  right (r),  body

syntax:

for (left of right)
  body

example:

{% highlight javascript %} for (let val of list) { f(val); } {% endhighlight %}

debugger (DebuggerStatement)

syntax:

debugger;

example:

{% highlight javascript %} debugger; {% endhighlight %}

Declarations

func-dec (FunctionDeclaration)

node attributes:
id,  body
node array attributes:
params
primitive attributes:
generator

syntax:

function id([param_1], [param_2], [..., param_3])
  body

example:

{% highlight javascript %} function f(x, y) { return x * y; } {% endhighlight %}

note:

A function declaration contrasts with a function expression (func-exp).

var-decs (VariableDeclaration)

node array attributes:
declarations (decs)
primitive attributes:
kind

syntax:

var declaration_1[, declaration_2, ..., declaration_n]

example:

{% highlight javascript %} var x = 1, y = 2; {% endhighlight %}

note:

Each declaration is a variable declarator (var-dec).

var-dec (VariableDeclarator)

node attributes:
id,  init

syntax:

id = init

example:

{% highlight javascript %} var x = 2 {% endhighlight %}

Expressions

this (ThisExpression)

example:

{% highlight javascript %} this {% endhighlight %}

super (Super)

example:

{% highlight javascript %} super(x, y) {% endhighlight %}

arr (ArrayExpression)

node array attributes:
elements (els)

[element_0, element_1, ..., element_n]

examples:

{% highlight javascript %} [1, 2, 3] [] {% endhighlight %}

obj (ObjectExpression)

syntax:

{
  property_1,
  property_2,
  ...,
  property_n

examples:

{% highlight javascript %} {a: 1, b: 2} {} {% endhighlight %}

func-exp (FunctionExpression)

node attributes:
id,  body
node array attributes:
params
primitive attributes:
generator

syntax:

function [id]([param_1], [param_2], [..., param_3])
  body

example:

{% highlight javascript %} let f = function(x, y) { return x * y; } {% endhighlight %}

note:

A function expression contrasts with a function declaration (func-dec).

arrow (ArrowFunctionExpression)

node attributes:
id,  body
node array attributes:
params
primitive attributes:
generator,  expression (exp)

syntax:

([param_1], [param_2], [..., param_3]) => body

example:

{% highlight javascript %} (x, y) => x * y {% endhighlight %}

seq (SequenceExpression)

node array attributes:
expressions (exps)

syntax:

expression_1, expression_2, ..., expression_n

example:

{% highlight javascript %} a, b, c {% endhighlight %}

yield (YieldExpression)

node attributes:
argument (arg)

syntax:

yield argument

example:

{% highlight javascript %} yield x {% endhighlight %}

unary (UnaryExpression)

node attributes:
argument (arg)

syntax:

operatorargument

examples:

{% highlight javascript %} +x typeof x {% endhighlight %}

bi (BinaryExpression)

node attributes:
left (l),  right (r)
primitive attributes:
operator (op)

syntax:

left operator right

example:

{% highlight javascript %} x === z {% endhighlight %}

assign (AssignmentExpression)

node attributes:
left (l),  right (r)
primitive attributes:
operator (op)

syntax:

left operator right

example:

{% highlight javascript %} (y = 2) {% endhighlight %}

update (UpdateExpression)

node attributes:
argument (arg)
primitive attributes:
operator (op),  prefix

syntax:

argumentoperator

or, if prefix

operatorargument

examples:

{% highlight javascript %} ++x x-- {% endhighlight %}

logic (LogicalExpression)

node attributes:
left (l),  right (r)
primitive attributes:
operator (op)

syntax:

left operator right

example:

{% highlight javascript %} x && y {% endhighlight %}

cond (ConditionalExpression)

node attributes:
test,  consequent (then),  alternate (alt, else)

syntax:

test ? consequent : alternate

example:

{% highlight javascript %} x % 2 ? "odd" : "even" {% endhighlight %}

new (NewExpression)

node attributes:
callee
node array attributes:
arguments (args)

syntax:

new callee(argument_1, argument_2, ..., argument_n)

example:

{% highlight javascript %} new Date(2011, 11, 11) {% endhighlight %}

call (CallExpression)

node attributes:
callee
node array attributes:
arguments (args)

syntax:

callee(argument_1, argument_2, ..., argument_n)

example:

{% highlight javascript %} f(1,2,3) {% endhighlight %}

member (MemberExpression)

node attributes:
object (obj),  property (prop)
primitive attributes:
computed

syntax:

object.property

example:

{% highlight javascript %} Math.PI {% endhighlight %}

template-literal (TemplateLiteral)

node array attributes:
quasis,  expressions (exps)

tagged-template-exp (TaggedTemplateExpression)

node attributes:
tag,  quasi

Clauses

switch-case (SwitchCase)

node attributes:
test
node array attributes:
consequent (then)

syntax:

case test | default :
  consequent

examples:

{% highlight javascript %} case 1: z = 'one'; break; default: z = 'two' {% endhighlight %}

catch (CatchClause)

node attributes:
param,  body

syntax:

catch (param)
  body

example:

{% highlight javascript %} catch (e) { console.error(e.message); } {% endhighlight %}
