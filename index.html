grasp

install npm install -g grasp

Search and replace your JavaScript code based on its structure rather than its text

Skip to Demo

grasp is a command line utility that allows you to search and replace your JavaScript code - but unlike programs such as "grep" or "sed", it searches the structure behind your code (the abstract syntax tree), rather than simply the text you've written - this allows for much more powerful searches.

Search using CSS style selectors 
$ grasp 'if.test [op=&&]' a.js
2:    if (x && f(x)) { return x; }
9:  if (xs.length && ys.length) {
14:  if (x < 2 && list[x]) {
Search using JS code w/ wildcards 
$ grasp -e 'return __ + __' b.js
3:  if (x < 2) { return x + 2; }
19:    return '>>' + str.slice(2);
21:  return f(z) + x;
Make complex replacements 
$ cat c.js
f(x < y, x == z);
$ {% raw %}grasp bi --replace '{{.r}}+{{.l}}' c.js{% endraw %}
f(y+x, z+x);
